# Medical Hybrid Encryption System (Qader Cyber Edition)

A professional medical data security suite developed as a graduation project. 

## 🛡️ Features
- **Hybrid Encryption:** Combines RSA-2048 (Asymmetric) and AES-256-GCM (Symmetric).
- **Advanced Key Derivation:** Uses PBKDF2 with SHA-256 and 100,000 iterations for maximum security.
- **Invisible Key Storage:** Implements an "Out-of-Band" secret key exchange by hiding security metadata as system files.
- **Modern UI:** Built with PyQt6 featuring a multi-page dashboard.
